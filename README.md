# **VSC A11Y**
### About us
This is a project made by Filip, Leon and Mateo.  
We are from a Croatian informatics club called **[C.H.A.O.S. Josipovac](https://www.facebook.com/c.h.a.o.s.josipovac/)**.

## What it's about
A11Y is an abbreviation for **accessibility** beacuse there are 11 letters between a and y.  
Its a VS Code accessibility extension with **daltonism-friendly themes**, **customizable syntax highlighting**, and **readability improvements** for **low-vision users**.
### Our project offers the following:  
|                Feature                 |    Explanation
|----------------------------------------|------------------
| Daltonist-friendly syntax highlighting | highlights errors and warnings with adjustable colors
| Daltonist-friendly themes              | Built-in color themes for most common types of daltonists
| Adjustable minimum terminal font size  | QOL feature so you dont have to dig around in vsc settings

## How to install
Just simply install the exstension.  
Its plug-n-play!

## How to use
When you install the exstension you immediately get the themes with it.  
The syntax highlighting is also enabled immediately and you get a new sidebar window to adjust the colors.  
The same window is where you can adjust the minimum terminal font size.
